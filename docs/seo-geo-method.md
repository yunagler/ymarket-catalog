# שיטת SEO + GEO של וואי מרקט — הפלייבוק המנצח

> **מקור:** השיטה הוגדרה בעבודה על עמודי המסכות/כפפות ועבדה מצוין (דירוגים גבוהים). שוחזרה מעמוד-הזהב `nitrile-black-perfect` ואומתה ע"י יובל (03/07/2026). **זו השיטה הרשמית לכל עמוד מוצר — לא לסטות ממנה.**

## למה השיטה הזו מנצחת
מנצחת בשני מנועים בו-זמנית: **גוגל** (תוכן עמוק + rich snippets) ו-**מנועי AI/GEO** (ChatGPT/Perplexity/Gemini — מצטטים תוכן מובנה מוכוון-תשובה). לא כמות — **עומק + מבנה + schema**.

## 1. מבנה התוכן (H3 קבועים, מוכווני-תשובה)
כל עמוד מוצר = ~25,000-30,000 תווי טקסט, מסודר ב-H3 בסדר הזה:
1. **תיאור** — מה זה, עובדתי, מדויק (חומר, מידות, אריזה — רק מה שידוע)
2. **השוואה** — "X מול Y מול Z — מה ההבדל?" ← **הסעיף שמנועי-AI הכי מצטטים**. טבלה + פסקאות
3. **"למה דווקא X"** — הזווית הייחודית של המוצר (למה שחור / למה תאית / למה מידה זו)
4. **"למי מתאים"** — סגמנטים (מוסכים / מטבחים / חברות ניקיון / מרפאות...)
5. **"איך בוחרים"** — מדריך החלטה (מידה / עובי / סוג מתקן / כמות)
6. **"למה לקנות מוואי מרקט"** — מהירות (48ש') · דיוק · מחיר סיטונאי · זמינות

## 2. Schema עשיר (זה נותן את ה-rich snippets = כוכבים בגוגל)
- `Product` (שם, תיאור, brand, mpn/gtin)
- `Offer` **לכל וריאציה** (מחיר, זמינות, מטבע)
- **`AggregateRating` + `Review` (Person)** — ⚠️ **חוות דעת אמיתיות בלבד** (מדירוגי לקוחות/רווחית — אישר יובל שהן אמיתיות). אסור לפברק
- `MerchantReturnPolicy` + `OfferShippingDetails` (מדיניות החזרות + משלוח)
- `FAQPage` (6-7 שאלות) + `BreadcrumbList`

## 3. עקרונות GEO (ציטוטיות במנועי-AI)
- **תשובות ישירות** בתחילת כל סעיף (המשפט הראשון עונה על השאלה)
- **טבלת השוואה** (מנועי-AI אוהבים טבלאות)
- **מבנה שאלה→תשובה** (FAQ + כותרות-שאלה)
- **עובדות חד-משמעיות** (מספרים, מידות, לא "איכותי מאוד")

## 4. On-page טכני
- Title 50-60 תווים עם מילת-המפתח + מותג · Meta 120-158 · H1 = שם המוצר
- **slug אנגלי** · Title/H1/תוכן/alt **בעברית** עם מילות מפתח ([[feedback_url_language_policy]])
- קישורים פנימיים בטקסט (מוצרים קשורים + קטגוריה + כתבות)
- canonical + og + image alt תיאורי

## 5. גרדריילים קשיחים (חוק)
- **עובדות אמת בלבד** — אסור להמציא מוצר/מפרט/תקן/"חדש"
- **רגולטורי-זהיר** — בלי "מאושר"/"תקן X"/"אמ״ר" בלי מסמך; מותר "מסייע"/תיאור עובדתי
- **בלי שמות מתחרים/לקוחות** בתוכן ציבורי
- **בלי "אנחנו קטנים"** — אמינות/מהירות/דיוק/זמינות
- **חוות דעת = אמיתיות בלבד** (Google policy + חוק הגנת הצרכן)

## ⚠️ הלקח שנצרב (מה לא לעשות)
- **לא כמות במקום עומק** — עמוד דל של 700 תווים ≠ השיטה. חובה 25K+ מובנה.
- **לא לשנות מבנה (איחוד וריאציות/מיגרציה) בלי למדוד דירוגים לפני-אחרי + לשמר כותרות** ([[feedback_variant_consolidation_seo_risk]])
- **לא geoContent גנרי דל** — זו הסטייה שהחלישה. השיטה = המבנה המלא למעלה.
